# Bakırçay Fall-2024 Game Development  
### Midterm Project  

## Project Overview  
This project is part of the Fall 2024 Game Development course at Bakırçay University. The goal of the midterm phase is to build the foundation of a 3D matching game using Unity. This repository contains a Unity project that fulfills all midterm requirements.  

### Midterm Objectives:  
1. **Basic Game Setup**:  
   - A central game area where 3D objects are scattered randomly.  
   - A circular placement area at the bottom of the screen for object placement.  
   
2. **Object Interaction**:  
   - **Object Movement**: Players can pick up and drag objects using touch or mouse input.  
   - **Object Placement**: Only one object can be placed in the designated placement area at a time.
